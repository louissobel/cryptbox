cryptbox
=========

858 Project for encryption in dropbox

Getting set up
-------------

requires FUSE:

 - [MacFUSE](https://code.google.com/p/macfuse/)
 - [Linux](http://fuse.sourceforge.net/)

Then:

```
virtualenv env
source env/bin/activate
pip install -r requirements.txt
```
